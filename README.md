# Goliath CSS
Simple StyleSheets for JavaFX
---
## Sheets
Goliath-Magma.css
Goliath-Light.css
Goliath-Mint.css
---
## Adding

```
Import the jar file and do:
'APP_SCENE.getStylesheets().add("goliath/css/Goliath-Magma.css");'
```
---
## Compatibility
Excecuting programs not in Windows's `path` must have `cmd /c` in order to execute.
